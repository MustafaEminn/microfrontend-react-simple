# A Micro Frontend Basket Project with React, Webpack Module Federation and Event Bus.
### How to run the project?

1. Open 3 terminal.

First Terminal
```
cd shell
npm i && yarn
yarn webpack server
```
Second Terminal
```
cd product
npm i && yarn
yarn webpack server
```

Third Terminal
```
cd basket
npm i && yarn
yarn webpack server
```

Open localhost:3001 when all apps running.
